== Description ==

The Wordpress copyscape plugin allows you to check posts and pages for duplicate content issues via the Copyscape API.  
A link is added on the post and page listing page next to the "Quick Edit" link that allows you to check for Copyscape issues.
Links and examples of any Copyscape issues are returned.

== Installation ==
1. Upload all plugin files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
